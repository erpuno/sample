SAMPLE: Move from Elixir to N2O
===============================

TL;DR Simple sample how to use N2O from Elixir.

Features
--------

* Usage Example of N2O, KVX, NITRO
* Mnesia and RocksDB support out of the box
* Size is 100 LOC

Prerequisites
---

* cmake (rocksdb)

Run
---

```
$ mix deps.get
$ mix compile
$ iex -S mix
```

Then open `http://localhost:8002/app/index.htm`
